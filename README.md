# Qu-n-L-Chi-Ti-u-C-Nh-n
HỆ THỐNG QUẢN LÝ CHI TIÊU CÁ NHÂN

Giới thiệu

Đây là dự án đồ án môn học xây dựng hệ thống quản lý chi tiêu cá nhân.
Hệ thống giúp người dùng quản lý các khoản thu nhập, chi tiêu, phân loại danh mục và theo dõi tình hình tài chính cá nhân một cách rõ ràng và hiệu quả.

Chức năng chính
 • Đăng ký và đăng nhập tài khoản
 • Quản lý danh mục thu nhập và chi tiêu
 • Thêm, sửa, xoá các khoản thu nhập
 • Thêm, sửa, xoá các khoản chi tiêu
 • Xem thống kê và báo cáo tài chính

Công nghệ sử dụng
 • C# WinForms (xây dựng giao diện ứng dụng)
 • Microsoft SQL Server (quản lý và lưu trữ dữ liệu)
 • Microsoft Visual Studio (môi trường phát triển ứng dụng)

Hướng dẫn chạy hệ thống
 1. Tải source code từ GitHub.
 2. Mở file .sln bằng Microsoft Visual Studio.
 3. Import file cơ sở dữ liệu vào Microsoft SQL Server.
 4. Chạy chương trình để sử dụng hệ thống.
